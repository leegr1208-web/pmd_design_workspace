Outlook [PMD] 디자인캘린더 자동 연동 사용 중

새 태스크를 등록하면 [PMD] 디자인캘린더에 일정이 자동으로 생성됩니다 (Microsoft Graph 직접 연동). 별도의 Power Automate 흐름이나 수동 버튼은 필요 없습니다.

⚠ 자동 생성은 Calendars.ReadWrite 관리자 동의가 완료돼야 동작합니다. 동의 전이면 태스크는 저장되지만 캘린더 생성은 조용히 건너뜁니다.
